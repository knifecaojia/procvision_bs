# 后端源码目录

基于 RuoYi-Vue 分离版开发的 Spring Boot 后端项目。

## 模块说明

*   **ruoyi-admin**: 后台服务入口，包含 Controller 层
*   **ruoyi-business**: **核心业务模块**，包含 MOM 对接、工艺管理、工单管理、检测结果管理等逻辑
*   **ruoyi-common**: 通用工具模块，包含工具类、常量、注解等
*   **ruoyi-framework**: 框架核心模块，包含 Spring Security、Web 配置等
*   **ruoyi-system**: 系统模块，包含用户、角色、菜单等基础权限管理逻辑
